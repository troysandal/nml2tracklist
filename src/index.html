Traktor NML To Human Readable Playlist

You can use this tool to turn a Traktor NML file into a human readable format that is ready to post to MixCloud, SoundCloud or wherever you want to give people the list of tracks in your set. Use the Format String to modify what each entry in the playlist looks like. There are multiple fields you can choose from to get the look you want.
